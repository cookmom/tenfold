Islamic Geometric Patterns

An interactive exploration of the mathematics behind Islamic art

For over a thousand years, Islamic artists have created geometric patterns of extraordinary complexity using only two tools: a compass and a straightedge. The patterns that emerge from these simple instruments — stars, rosettes, interlacing bands — decorate mosques, palaces, and manuscripts across the Islamic world.

This guide explores the mathematical foundations of these patterns, from the circle grids that generate them to the quasicrystal tilings that anticipated modern physics by five centuries. Each section includes interactive demonstrations — drag the sliders to see how parameters shape the pattern.

The Circle Grid

Every Islamic geometric pattern begins with a grid of overlapping circles. The pattern doesn't exist in any single circle — it emerges at the intersection points where circles overlap. Connect adjacent intersections on each circle, and a star pattern appears.

This is the fundamental insight: the pattern is a consequence of pure geometry, not a design imposed upon it. 

import { circleGrid, emergentPattern, toSVG } from 'tenfold'

const centers = circleGrid({ x: 0, y: 0, w: 400, h: 400, size: 40, type: 'hex' })
const lines = emergentPattern(centers, 20)
const svg = toSVG(lines, { width: 400, height: 400 })

document.body.innerHTML = svg

The hexagonal grid produces 6-fold symmetry — the basis of patterns found in Moroccan zellige tilework. The square grid produces 8-fold symmetry, characteristic of Persian and Central Asian designs.
